Current Affairs Generator
Three methods to generate exam-ready current affairs articles → Firestore

🤖 Auto Daily Generation

Runs automatically every day at 6:30 AM IST via GitHub Actions. Generates articles for all 17 categories across UPSC, UPPCS, BPSC, SSC CGL & UPSSSC. You can also trigger it manually below.
Setup required: Add these secrets to your GitHub repo (Settings → Secrets → Actions):
• GROQ_API_KEY — your Groq API key (free at console.groq.com)
• FIREBASE_API_KEY — from firebase-config.js
• FIREBASE_PROJECT_ID — current4exams
